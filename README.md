# 基于MATLAB语音信号处理（0-9语音识别，GUI）

#### 介绍
本设计为基于MATLAB的HMM语音信号识别，可以识别0-9十个阿拉伯数字，带有一个丰富的人机交互GUI界面。算法流程为：显示原始波形图……显示语音结束处放大波形图……显示短时能量……设置门限……开始端点检测……，也可以通过添加噪声，对比加噪后的识别准确率。后续可二次开发，做成九宫格形式，做一个电话拨号音识别。
